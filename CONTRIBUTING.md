# Contributing

Merci de contribuer ! Avant de soumettre une PR :
1. Fork le dépôt et travaille sur une branche `feature/<nom>`.
2. Respecte les conventions de commit (Conventional Commits).
3. Lance `pre-commit` localement, et les tests `pytest`.
4. Ouvre une Pull Request depuis la branche feature vers `dev` (ou `main` si pas de dev).
